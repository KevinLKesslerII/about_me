Pokemon Information

Serebii
Serebii has everything you need to know Pokemon related. Pokemon by number, Games, Episodes, Shorts, Movies, Trading Cards, etc.
Pokemon Showdown
More for the competetive player, this site allows you to build teams and battle with them online against other players. Test out sets and strategies!
Bulbapedia
This site is similar to Serebii in content but it is presented in a wiki style format.

Game Development

GameDev.tv Courses
Has courses related to game development on game engines such as Godot 4, Unity, and Unreal Engine 5.
Unity Learn
Has courses related to development for the Unity engine.
Unreal Engine Tutorials
Game Development courses and tutorials for Unreal Engine

Computer Programming

Boot.dev
Courses to help you become a backend developer while making games! Courses in Python, SQL, Git, HTTP, JS, etc.
Code With Mosh
Has courses for full-stack development. From languages to frameworks, this site has it all.
Code Wars
A training tool that has community led coding challenges for multiple programming languages. You are ranked in a dojo like system (9th kyu - 1st dan) based on what challeneges you complete.
